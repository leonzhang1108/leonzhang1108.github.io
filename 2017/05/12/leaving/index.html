走了

2017.05.12 Leon Zhang  热度 ℃

  离职要比面试要难

  本来礼拜一就打算讲了，然后看着大家努力的工作气氛，硬是没敢说出口。第二天鼓起勇气，说怎么也得今天给讲了，结果晚上前端分享上礼拜去eleme听课成果，结果还是怂了。礼拜三晚上，坐椅子上续了一下午的勇气，最后终于拔地而起拉峰哥进了小黑屋，说我准备离职。过程很官方，感觉蛮尴尬，可能是觉得我不留下也没多可惜，但好歹试图挽留。这两年被yk带入前端，着重还是配合着做业务，说认真学习其实基本是没有的。每次去听课都是云里雾里，像是被拔苗助长，专业名词累计了很多，然而并没有什么软用。现在面试除了基础，运用框架的程度和时间是第二大标准，光自己做了个NoteAnywhere有个鸟用。

  礼拜三讲完离职，回家yk微信说明天跟我聊聊。于是今天又一次被拉进小黑屋。做了两道题，在面试了久了之后也不是很怵。我离职理由说是想花更多时间学习，yk的solution是给私活儿，或者尽量能在工作项目中运用起来，但是更主要的理由并没敢说出来。here it is.

  1. 不太喜欢玩笑过度的工作氛围。在很严肃的讨论需求细节时，当我很严肃的寻求解决方案，希望对方配合，对方：我不～ 这挺可爱，但是工作中就是个傻x。但是还要忍，脚趾头想也知道一旦骂出来会变得更尴尬。就这样，配合着开了10分钟玩笑，解决了本来20秒就能解决的问题。为了不做一个working bitch，要牺牲挺多时间的。。

  2. 过度攀比的学习气氛。这点可能是自己脑子太敏感导致的。我认为讨论是人们进步的基石，合理的讨论甚至碰撞，是几个人大脑融合的过程。而我们所谓的讨论，就是装逼啊。。谦虚很重要，心态要放平。。奇葩说里，黄执中说的话能一字一句完全带着你走进他的持方，而总有人一字一句能把自己捧到天上去，像个穿着新衣的皇帝，赤膊着在百姓面前搔首弄姿。。不好不好。

  3. 太过谦和的领导。做错事，出了bug，leader们会抱怨但是从来不会骂。自己粗心出过很多次大bug，自己很愧疚，但是修好之后就不了了之了。我自己可能贱，希望被骂一下，有外力能给自己一个警钟，给自己一个做错事就要挨骂的条件反射，督促自己今后一切粗心的行为，争取不再犯。

  4. 坑爹的工作效率。其实我由于自己码代码快而bug总是来得始料未及，并没有什么资格说效率。。但是我觉得效率也可以靠练习获得的。而我看不惯的，是并没有追求效率的工作态度。上班tm打狼人杀。。。qq联系完全不鸟你。。总在deadline前夕开始工作。。。还有一个可能变相让大家觉得时间还多，可以玩一会儿再工作的事儿，是固定的叫外卖晚饭。一般有晚饭的加班日，回家基本上都能晚到打车能报销。而有晚饭的日子基本上是天天（我也没天天加班）。

  晚上打开oa写了离职报告。如果难熬，接下来的倒计时可能是最难熬的。面对平时欢声笑语的气氛会变得沉默，虽然离职是对自己好，但面对leader们和同事们会不舍。两年里来了很多人走了很多人，一直在的也有很多人，看到大家努力顶住了两年的辛苦中，也有自己的一份苦劳，会难过。但是看清楚自己，躲在温室里的鸟永远是家禽。yht容纳了当年连个p都不是的我，前端进阶过程中多少自己挖的坑被大家包容，多少犯的错被原谅，都在感恩这两个字当中。感恩行动做不来，那就心里默念500遍感谢yht。
